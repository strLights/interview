# 常见状态码

## 1XX 服务器接收请求

## 2XX 请求成功 如 200

## 3XX 重定向 如 302

## 4XX 客户端错误 如 404
## 5XX 服务端错误 如 500



##常见状态码

200 请求成功

301 永久重定向 （配合location，浏览器自动处理）
比如有个网站的域名到期了，而且不用了，我们就可以把老的域名定到新域名下

302 临时重定向 （配合location，浏览器自动处理）

比如百度一个网站，然后打开，检查代码时候，会发现这是个百度的地址，打开这个地址然后重定向到你要访问的地址。

304

比如访问之前一个打开的页面，一些css的js的链接请求，代码都没有改变，返回304，用浏览器缓存的即可，不用获取新的资源。

404 资源访问失败

403 没有权限

500 服务端错误

503 网管超时
